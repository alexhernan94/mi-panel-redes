# Términos de Servicio — itsbgart Analytics

**Última actualización:** 16 de julio de 2026

## 1. Descripción del servicio

itsbgart Analytics es una herramienta privada de analítica de redes sociales diseñada para uso personal de la creadora de contenido itsbgart. La aplicación permite visualizar métricas de rendimiento de contenido publicado en TikTok, Instagram y YouTube.

## 2. Uso permitido

- Esta aplicación es de uso exclusivo para el propietario de la cuenta autorizada
- No está diseñada para uso público ni comercialización a terceros
- El acceso está protegido por contraseña

## 3. Acceso a datos de TikTok

- La aplicación accede a datos de TikTok únicamente mediante la API oficial de TikTok for Developers
- Solo se accede a datos del propio usuario autenticado, nunca a datos de terceros
- Los datos se usan exclusivamente con fines analíticos y de mejora de estrategia de contenido

## 4. Limitación de responsabilidad

- La aplicación se proporciona "tal cual", sin garantías de disponibilidad continua
- No nos hacemos responsables de cambios en las APIs de terceros que afecten la funcionalidad
- Los análisis y recomendaciones generados por IA son orientativos, no garantizan resultados

## 5. Propiedad intelectual

- El contenido analizado pertenece al creador original
- Las métricas y datos pertenecen a las plataformas respectivas (TikTok, Instagram, YouTube)
- La aplicación y su código son propiedad privada del desarrollador

## 6. Modificaciones

Nos reservamos el derecho de modificar estos términos en cualquier momento. Los cambios entran en vigor inmediatamente tras su publicación.
